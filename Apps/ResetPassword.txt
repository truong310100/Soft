Bước 2: Reset khóa khi vào được cmd:
	Cách 1
		-> nhập notepad.exe 
		-> file -> open 
		-> tìm utiman rồi đổi tên thành utiman-old 
		-> tìm cmd sữa tên thành utiman rồi reset lại 
		-> vào chọn icon đồng hồ rồi gõ lusrmgr.msc 
		-> set lại pass
	Cách 2: 
		nhập net user
		-> chọn “user + …(tên user)... + * “ rồi nhập pass
		**note: nó sẽ không hiện pass, cứ nhập rồi enter rồi reset lại
